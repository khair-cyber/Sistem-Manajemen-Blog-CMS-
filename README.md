# Sistem-Manajemen-Blog-CMS-
Sistem Manajemen Blog (CMS) berbasis PHP &amp; MySQL   Proyek UTS Pemrograman Web Semester Genap 2025/2026. Aplikasi ini memungkinkan pengelolaan data Penulis, Artikel, dan Kategori Artikel dengan fitur CRUD (Create, Read, Update, Delete) yang berjalan secara asynchronous menggunakan fetch API.
